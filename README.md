# Financial Risk Analysis with Power BI

## Problem Statement
This project explores financial risk patterns in a sample dataset to understand repayment behavior and risk segmentation.

## Dataset
The dataset contains information on customer profiles, loan amounts, and repayment status.

## Tools Used
- Power BI
- Excel

## Key Questions
- Which customer segments show higher risk?
- Are there observable patterns in repayment behavior?

## Status
Project in progress.
